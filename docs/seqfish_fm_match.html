This Class aligns dots in a seqFISH experiment image containing fiducial markers, to a reference image containing only the fiducial markers

Parameters:

  • ro – pandas dataframe of reodout points with columns = (“x”, “y”, “z”, “int”) and index “hyb” denoting the hybridiztion in which the point was found.

  • ref – pandas dataframe of reference points with columns = (“x”, “y”, “z”, “int”)

  • ref_final – Optional- pandas dataframe of final reference points measured after readout hybridizations. Used for automatically setting parameters in the auto_set_parameters method.
