Welcome to seqfish_fm_match's documentation!
===========================================
This package provides fiducial marker matching alignment through the FMMatcher object,
which can align images from seqFISH experiments. It has been used in both the
`Nature <https://doi.org/10.1038/s41586-020-03126-2>`_ and `Science <https://www.science.org/doi/10.1126/science.abj1966>`_
DNASeqFISH+ papers.

Documentation on the FMMatcher object and its methods can be found on
the :doc:`api` page.
